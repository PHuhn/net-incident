Before Updating
There aren't currently any changes needed before moving between these versions.
During the Update
__ Run ng update @angular/core@12 @angular/cli@12 which should bring you to version 12 of Angular.
__ Angular now requires TypeScript 4.2. ng update will update you automatically.
__ IE11 support has been deprecated. Find details in the RFC for IE11 removal.
__ You can no longer use Angular with Node.js version 10 or older
After the Update
There aren't currently any changes needed after moving between these versions.
